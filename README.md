# 📏 今日长度 (Today Length)

✨ 一个基于 AstrBot 的趣味娱乐插件，专为群聊活跃气氛设计！✨

## 📖 插件介绍

用户发送指令 `今日长度` 后，机器人会随机生成一个 1-20 之间的数字（保留两位小数）。
该插件具备 **"每日锁定"** 功能：同一个用户在同一天内测得的数值是固定的，重复查询将返回第一次的结果。

## 🚀 功能特性

* 🎲 **随机生成**：1.00cm - 20.00cm 随机数值。
* 🔒 **每日锁定**：数值每日凌晨自动刷新，当天重复测试结果一致。
* 👥 **完美适配**：支持私聊、群聊环境，并在群聊中自动 @ 触发用户。
* 💾 **持久化存储**：使用 JSON 文件保存数据，重启容器记录不丢失。

## 🛠️ 安装方法

### 手动安装

1. 进入你的 AstrBot 插件目录：
   ```bash
   cd AstrBot/data/plugins
   ```
2. 克隆本仓库：
   ```bash
   git clone https://github.com/yoyoakalin/today_length.git
   ```
3. 重启 AstrBot 或使用插件管理功能加载插件。

### 通过 AstrBot 插件市场安装

在 AstrBot 插件市场中搜索 "today_length" 并安装。

## 📝 使用方法

在聊天中发送以下指令即可：

```
今日长度
```

机器人会回复您的"今日长度"数值。

## ⚙️ 配置项

无需额外配置，插件会自动工作。
