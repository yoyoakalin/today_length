# 📏 今日长度 (Today Length) - Pro 版

✨ 基于 AstrBot 的高性能趣味娱乐插件，专为活跃群聊气氛而生！ ✨

此版本已针对高并发环境重构，采用 SQLite 数据库存储，支持完全异步 I/O，彻底解决 JSON 存储在高并发下的覆盖与损坏风险。

## 📖 插件介绍

用户发送指令 `今日长度` 后，机器人会随机生成一个 1.00cm - 30.00cm 之间的数值。该插件具备 "每日锁定" 功能：同一用户在同一天内测得的数值是固定的，重复查询将返回一致的结果，次日凌晨自动重置。

## 🚀 核心特性

* 🎲 **精准随机**：支持 1.00cm - 30.00cm 随机数值。
* 🔒 **每日锁定**：数值每日凌晨自动刷新，逻辑严密，即使重启插件也不会失效。
* � **实时排行**：新增 长度排行 指令，支持分页显示，避免在大群中刷屏。
* �️ **SQL 驱动**：使用 SQLite 异步数据库，具备 ACID 特性，数据存储稳如磐石。
* ⚡ **异步优化**：采用 aiosqlite，网络请求与数据库写入均不阻塞机器人主进程。
* 👥 **全平台兼容**：完美支持 QQ、Telegram、Discord 等多种 ID 体系，不再因非数字 ID 崩溃。
* 📏 **规范代码**：严格遵循 PEP 8 编码规范，逻辑清晰，易于二次开发。

## � 安装方法

### 1. 安装依赖 (必要)

由于采用了异步数据库驱动，请在你的 AstrBot 运行环境中执行：

**pip:**
```bash
pip install aiosqlite
```

**如果你使用 uv 管理包：**
```bash
uv pip install aiosqlite
```

### 2. 下载插件

进入你的 AstrBot 插件目录并克隆本仓库：

```bash
cd AstrBot/data/plugins
git clone https://github.com/yoyoakalin/today_length.git
```

然后重启 AstrBot 即可自动加载。

## 📝 使用方法

在聊天中发送以下指令：

| 指令 | 说明 |
|------|------|
| 今日长度 | 获取你今天的随机长度（每天仅限生成一次新结果） |
| 长度排行 | 查看今日长度排行榜（自动分页，每页 15 人） |

## ⚙️ 技术规格

* **开发语言**：Python 3.10+
* **存储引擎**：SQLite 3 (data.db)
* **异步库**：aiosqlite
* **并发策略**：双重检查锁定 (Double-Checked Locking) + 异步互斥锁
