# AstrBot 今日长度插件 (Database Edition) 🏆

基于 AstrBot 框架开发的趣味娱乐插件。通过随机算法生成用户当日的“长度”，支持全员排行榜，并采用 SQLite 数据库进行高性能数据存储。

## ✨ 核心特性

- **数据持久化**：采用 SQLite 数据库，即便机器人重启，今日记录也不会丢失。
- **高性能异步**：所有数据库 I/O 均在独立线程池运行，绝不卡顿机器人主进程。
- **智能昵称识别**：自动获取群名片或 QQ 昵称，榜单显示更直观。
- **并发锁机制**：内置数据库事务保护，无惧百人群聊同时触发。
- **零依赖安装**：仅使用 Python 标准库，插件拖入即用。

## 🛠️ 安装方法

1. 进入你的 AstrBot 插件目录：`vendor/plugins/`。
2. 创建文件夹 `today_length`。
3. 将 `main.py` 放入该文件夹。
4. 在机器人控制面板点击 **“插件管理” -> “重载插件”**。

## 🎮 使用指令

| 指令 | 别名 | 描述 |
| :--- | :--- | :--- |
| `/今日长度` | 无 | 随机生成或查询你今天的长度（每日锁定一次） |
| `/长度排行` | `/全员榜` | 查看今日群内所有已测成员的降序排行榜 |

## 📂 文件结构

- `main.py`: 插件核心逻辑。
- `data/today_length/lengths.db`: (自动生成) 存储用户数据的 SQLite 数据库文件。

## ⚠️ 开发说明

- **ID 兼容性**：代码已针对非数字 ID（如 Telegram/邮箱）进行适配，不会因类型转换崩溃。
- **消息限长**：排行榜采用 15 人/组的分片发送模式，有效防止触发平台风控或刷屏。
